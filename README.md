#DỰ ÁN THIẾT KẾ TRANG WEB THEO MẪU 

##Giới thiệu
Đây là một trang web tĩnh được xây dựng bằng **HTML, CSS và JavaScript** nhằm mục đích:
- Định dạng bố cục theo 3 phần: header footer và nội dung.
- Nội dung sẽ được chia thành 2 cột chính.
- Bên trái là các tùy chọn, bên phải là nội dung của từng tùy chọn đó.

## 🛠️ Công nghệ sử dụng
- **HTML5**: Tạo cấu trúc nội dung.
- **CSS3**: Thiết kế giao diện, màu sắc, bố cục.
- **JavaScript (ES6)**: Thêm hiệu ứng động.

##Cấu trúc thư mục
MyAssignment/
├── index.html			#Trang chính
├── style.css			#Định dạng cho trang chính
├── script.js			#File xử lý hiệu ứng, hành động cho trang chính
├── assets/
│ 	├── images/			#Hình ảnh minh họa
│ 	├── css/			#Các file css bổ sung
│ 	└── html/			#Các file html bổ sung
│ 
└── README.md			#Tài liệu bổ sung

##Nội dung trang web 
-Giới thiệu sơ bộ về đặc điểm, môi trường sống và nguồn thức ăn của các loài động vật.
-Động vật được chia thành 4 loại dựa trên nguồn thức ăn của chúng: 
	+ Động vật ăn cỏ.
	+ Động vật ăn thịt.
	+ Động vật ăn tạp.
	+ Động vật ký sinh.
-Bảng so sánh các loài động vật này.
